# 🎓 Student Grade Tracker

## 📌 Project Overview
Student Grade Tracker is a Java console application developed as part of the CodeAlpha Java Programming Internship. It helps users enter student names and marks, then calculates and displays the average, highest, and lowest marks along with a summary report.

## ✨ Features
- Enter student names
- Enter student marks
- Store data using Arrays
- Calculate average marks
- Find highest marks
- Find lowest marks
- Display a summary report

## 🛠️ Technologies Used
- Java
- VS Code
- JDK 8+

## 📂 Project Structure
```
StudentGradeTracker/
│── StudentGradeTracker.java
│── README.md
```

## ▶️ How to Run

1. Clone the repository
2. Open the project in VS Code
3. Compile the program

```bash
javac StudentGradeTracker.java
```

4. Run the program

```bash
java StudentGradeTracker
```
